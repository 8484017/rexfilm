Но я возьму только один для вас, всегда используйте === и ! == вместо == и ! = 

   alert   (      ==     // false  > 
alert   (  0     ==     ''  );     // true  
alert   (  0     ==   '0'  );     // true   


  ==  не является транзитивным. Если вы используете  === , это даст false для
все эти утверждения, как ожидалось.
